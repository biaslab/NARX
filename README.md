# NARX

Custom ForneyLab.jl factor node for a nonlinear autoregressive model with exogenous input.

### Usage

The node can be added to ForneyLab by running 

```julia
] dev git@github.com:wmkouw/NARX.git
```

in the REPL and then import

```julia 
using ForneyLab
using NARX
```
